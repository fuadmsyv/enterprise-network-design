# Layihə Təsviri

## Ümumi məlumat
Bu layihə müəssisə şəbəkəsinin qurulmasını əhatə edir. Məqsəd VLAN texnologiyası əsasında şəbəkənin segmentasiyası və idarə olunmasını təmin etməkdir.

## Layihənin məqsədi
- Şəbəkənin təhlükəsizliyini artırmaq
- Trafiki optimallaşdırmaq
- İdarəetməni asanlaşdırmaq

## İstifadə olunan texnologiyalar
- VLAN
- Inter-VLAN Routing
- Cisco Router və Switch

## İstifadə olunan alətlər
- Trello – tapşırıqların idarə olunması
- Jira – problemlərin izlənməsi
- Git – versiya nəzarəti

## Nəticə
Layihə nəticəsində müxtəlif VLAN-larda yerləşən istifadəçilər bir-biri ilə router vasitəsilə əlaqə qura bilir.
