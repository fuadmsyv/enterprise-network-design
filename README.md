# enterprise-network-design
# Müəssisə Şəbəkə İnfrastrukturunun Qurulması Layihəsi

## 📌 Layihə haqqında
Bu layihə müəssisə üçün miqyaslana bilən və təhlükəsiz şəbəkə infrastrukturunun layihələndirilməsi və qurulmasını əhatə edir. Layihə çərçivəsində VLAN əsaslı şəbəkə strukturu yaradılmış və router vasitəsilə VLAN-lar arası əlaqə təmin edilmişdir.

---

## 🎯 Məqsədlər
- VLAN əsaslı şəbəkənin qurulması
- Şəbəkənin segmentasiyası və təhlükəsizliyin artırılması
- Inter-VLAN routing konfiqurasiyası
- Şəbəkənin səmərəli idarə olunması

---

## 🛠 İstifadə olunan texnologiyalar
- VLAN (Virtual Local Area Network)
- Routing (Statik və ya Inter-VLAN routing)
- Cisco şəbəkə cihazları

---

## 📊 Layihə idarəetməsi
Layihə idarəetməsi aşağıdakı alətlər vasitəsilə həyata keçirilmişdir:

- Trello – tapşırıqların idarə olunması
- Jira – problemlərin və tapşırıqların izlənməsi
- Git – versiya nəzarəti və dəyişikliklərin idarə olunması

---

## 🌐 Şəbəkə topologiyası
Şəbəkə topologiyası və diaqramlar `/topology` qovluğunda yerləşir.

---

## ⚙️ Konfiqurasiyalar
Router və switch konfiqurasiyaları `/configs` qovluğunda saxlanılır.

---

## 📄 Sənədləşmə
Layihə ilə bağlı əlavə sənədlər `/docs` qovluğunda mövcuddur:
- IP ünvanlama planı
- VLAN planı
- Layihə izahı
